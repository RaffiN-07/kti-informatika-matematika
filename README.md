Aplikasi berbasis web ini merupakan implementasi praktis dari konsep **Geometri Lingkaran** untuk menghitung parameter mekanik pada kendaraan, khususnya hubungan antara jari-jari ban, jumlah putaran, dan jarak tempuh. Proyek ini disusun sebagai bagian dari **Karya Tulis Ilmiah (KTI)** bidang studi Informatika & Matematika.



## 🚀 Fitur Utama

* **📏 Kalkulator Jarak Tempuh**: Menghitung total jarak yang dilalui kendaraan berdasarkan ukuran ban dan intensitas putaran.
* **🔄 Kalkulator Intensitas Putaran**: Menentukan berapa kali ban berputar untuk mencapai jarak tertentu (sangat berguna untuk analisis keausan ban).
* **🌍 Konversi Unit Fleksibel**: Mendukung satuan *milimeter (mm), centimeter (cm), meter (m), kilometer (km),* hingga *inci (in)* dan *mil (mile)*.
* **📝 Langkah Penyelesaian Otomatis**: Menampilkan cara pengerjaan (step-by-step) secara matematis untuk keperluan edukasi.
* **📱 Responsive & Modern UI**: Tampilan *Glassmorphism* yang nyaman di mata dan kompatibel dengan perangkat mobile.

---

## 📑 Dasar Teori

Aplikasi ini menggunakan tiga pilar rumus utama lingkaran:

1.  **Keliling Lingkaran ($K$):**
    $$K = 2 \times \pi \times r$$
2.  **Jarak Tempuh ($d$):**
    $$d = n \times K$$
3.  **Jumlah Putaran ($n$):**
    $$n = \frac{d}{K}$$

> **Catatan:** Nilai $\pi$ (Pi) yang digunakan adalah pendekatan presisi tinggi $\approx 3,14159265$.
